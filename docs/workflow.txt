==================== STUDYCIRCLE MVP (BACKEND-FIRST) — ALL-IN-ONE FLOW ====================

USER ENTERS PLATFORM
│
└── AUTH SYSTEM
    │
    ├── Register (Student / Teacher)
    │     └── Password hashed → User stored → JWT issued
    │
    ├── Login
    │     └── Credentials verified → JWT issued
    │
    └── All Next Routes Protected
          └── Token → authMiddleware → req.user set

========================================================================================

DASHBOARD (After Login)
│
├── Student Actions
│     │
│     ├── Create Group
│     │     ├── Group created with joinCode
│     │     ├── createdBy = studentId
│     │     └── Creator becomes Admin Member
│     │
│     └── Join Group (Join Code)
│           ├── Group found by joinCode
│           └── Student added as Member

└── Teacher Actions
      └── Can participate only in groups where added

========================================================================================

ENTER GROUP (Main Learning Hub)
│
├── Membership Gate (Core Rule)
│     └── Only group members can access anything inside
│
├── RESOURCES MODULE (PDF / Links)
│     │
│     ├── Upload Resource
│     │     ├── Member check passes
│     │     ├── PDF stored in /uploads/resources
│     │     └── Resource saved in MongoDB
│     │
│     ├── View Resources
│     │     └── All members can list + download
│     │
│     └── Delete Resource
│           └── Only uploader (or admin)

├── QUESTIONS + ANSWERS MODULE (Doubts System)
│     │
│     ├── Student Posts Question
│     │     ├── Title + Description stored
│     │     └── Visible to all group members
│     │
│     ├── Students Answer (Peer Learning)
│     │
│     └── Teacher Answers (Verified Clarification)

========================================================================================

FINAL MVP CORE LOOP
│
Login/Register
 ↓
Student Creates or Joins Group
 ↓
Enter Group (Members Only)
 ↓
Share Resources + Ask Doubts
 ↓
Teacher Clarifies + Students Collaborate
 ↓
StudyCircle becomes a Learning Community
